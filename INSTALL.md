# ChatGPT Cleaner — Install Guide

## Quick Install (Chrome / Edge)

1. Download `chatgpt-cleaner-v1.0.0.zip`
2. Unzip it — you'll get a `dist` folder
3. Open Chrome and go to `chrome://extensions/`
4. Toggle **Developer mode** ON (top right)
5. Click **Load unpacked** (top left)
6. Select the `dist` folder
7. Done — the extension icon appears in your toolbar

### Usage
1. Go to [chatgpt.com](https://chatgpt.com)
2. Click the extension icon
3. It auto-scans your chats
4. Select chats → click **Delete**

---

## Install on Firefox

1. Open Firefox and go to `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Select `dist/manifest.json` from the unzipped folder

---

## Install on Edge

Same as Chrome — go to `edge://extensions/`, enable Developer mode, load unpacked.

---

## Notes

- **Keep the ChatGPT tab open** when deleting — the extension needs the page to be loaded
- Free tier: 10 deletes/day
- Pro tier: 75 deletes/day (get a license key from chatgptcleaner.com/pricing)
- After deletion, the page auto-refreshes to update the sidebar
